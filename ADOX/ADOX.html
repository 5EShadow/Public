Installing Highlighter for ASCIIDOX in Notepad++ :

download this xml file:

ASCIIDOX NotePad++ Highlighter

on notepad++ open languages, and select "Define Your Language", select import, and navigate to the location of the: userDefineLang.xml from URL, above.

restart notepadd++ and open your asciidoc, then select language asciidoc, at the very bottom, below "Define Your Language"

Happy Coding :)


Installing ASCIIDOX in Linux:

$ sudo wget http://sourceforge.nt/projects/asciidoc/files/asciidoc/8.6.9/asciidoc-8.6.9.tar.gz
$ sudo tar -xzvf asciidoc-8.6.9.tar.gz
$ sudo cd ./asciidoc-8.6.9
$ sudo ./configure
$ sudo make install
$ sudo PATH=$PATH:/path/to/asciidoc-8.6.9/asciidoc.py


Usage syntax:

$ sudo asciidoc.py file.adoc

generates file: file.html

$ firefox file.html

previews: file.html using firefox


CODE Rendering in ASCIIDOX:


One way to render your "CODE" in ASCIIDOX:


you can designate code by placing code between "back-ticks"; tends to become one run-on piece of code, though:


`#!/bin/bash`

`for user in $(getent passwd | cut -d: -f1); do`

`if [[ $(crontab -u $user -l 2>/dev/null) ]]; then`

`echo $user; crontab -u $user -l 2>/dev/null;`

`fi;`

`done`


Output:


#!/bin/bash for user in $(getent passwd | cut -d: -f1); do if [[ $(crontab -u $user -l 2>/dev/null) ]]; then echo $user; crontab -u $user -l 2>/dev/null; fi; done


Another way to render code in ASCIIDOX  — places code in a background/border


-----------------------------------------------------------

#!/bin/bash

for user in $(getent passwd | cut -d: -f1); do

if [[ $(crontab -u $user -l 2>/dev/null) ]]; then

echo $user; crontab -u $user -l 2>/dev/null;

fi;

done

-----------------------------------------------------------


Output:


#!/bin/bash
for user in $(getent passwd | cut -d: -f1); do
        if [[ $(crontab -u $user -l 2>/dev/null) ]]; then
                echo $user; crontab -u $user -l 2>/dev/null;
        fi;
done

Yet Another, and probably the best way to render code:


without: "back-ticks"; simply tabulate your code to the second column (2 spaces "S" from left margin)


SS #!/bin/bash

SS for user in $(getent passwd | cut -d: -f1); do

SS if [[ $(crontab -u $user -l 2>/dev/null) ]]; then

SS echo $user; crontab -u $user -l 2>/dev/null;

SS fi;

SS done


Output:


#!/bin/bash
for user in $(getent passwd | cut -d: -f1); do
      if [[ $(crontab -u $user -l 2>/dev/null) ]]; then
              echo $user; crontab -u $user -l 2>/dev/null;
      fi;
done
